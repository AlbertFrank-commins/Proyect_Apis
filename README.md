# Poke & Jokes (Consumo de APIs)

Estudiante: Albert Frank  
Matrícula: 2023-3794

## Descripción
Página web desarrollada con **HTML, CSS y JavaScript** que consume dos APIs públicas usando **fetch()**:

- **PokéAPI:** búsqueda de Pokémon por nombre o número, mostrando imagen, tipo y al menos 3 estadísticas.
- **JokeAPI:** muestra un chiste aleatorio al presionar un botón.

## Tecnologías usadas
- HTML
- CSS
- JavaScript (Fetch API)

## Cómo usar
1. Escribe el nombre o número de un Pokémon y presiona **Buscar** (o Enter).
2. Presiona **Traer chiste** para obtener un chiste aleatorio.
